系统架构设计

轻量化方案：Node.js 单服务进程 + SQLite + 零构建前端

架构概览

Browser → server.js → chat.db
server.js (~600行)
├ HTTP 路由（登录/注册/聊天/管理后台）
├ SSE 流式转发（代理 OpenAI 兼容 API）
├ Session/JWT 认证
├ 静态文件服务（无构建步骤）

public/（纯 HTML/CSS/JS，零依赖）
├ index.html — 聊天界面
├ admin.html — 管理面板
├ login.html — 登录/注册
├ style.css — 全局样式
├ app.js — 前端聊天逻辑
