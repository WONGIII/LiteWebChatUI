隐私政策

最后更新日期：2026年7月24日

1. 引言

欢迎使用 LiteChat（以下简称"本服务"）。我们深知隐私对您的重要性，并致力于保护您的个人信息。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的信息。

2. 信息收集

2.1 您主动提供的信息

当您使用本服务时，我们可能收集以下信息：

  • 账户信息：电子邮箱地址、密码（加密存储）
  • 对话内容：您与 AI 模型的对话记录
  • 配置信息：API 密钥、模型配置等

2.2 自动收集的信息

我们可能自动收集以下信息：

  • 设备信息：浏览器类型、操作系统、设备标识符
  • 使用数据：访问时间、页面浏览记录、功能使用情况
  • 日志数据：IP 地址、访问日志、错误日志

3. 信息使用

我们收集的信息将用于以下目的：

  • 提供、维护和改进本服务
  • 处理您的请求和交易
  • 发送服务通知和更新
  • 防止欺诈和滥用行为
  • 遵守法律法规要求

4. 信息存储与安全

4.1 数据存储

您的数据存储在您自部署的服务器上。如果您使用 Docker 部署，数据存储在容器内的 SQLite 数据库中。

4.2 数据安全

我们采取合理的技术和管理措施来保护您的个人信息，包括：

  • 密码使用 bcrypt 算法加密存储
  • 会话使用安全的 Cookie 机制
  • 定期进行安全审计和更新

5. 信息共享

我们不会将您的个人信息出售给第三方。但在以下情况下，我们可能共享您的信息：

  • API 提供商：当您使用 AI 对话功能时，您的对话内容会发送到您配置的 API 提供商（如 OpenAI、DeepSeek 等）
  • 法律要求：当法律法规要求时，我们可能需要披露您的信息
  • 权利保护：为保护我们的权利、财产或安全

6. 第三方服务

本服务可能包含指向第三方网站或服务的链接。我们对这些第三方的隐私政策不负责任。我们建议您在使用第三方服务前仔细阅读其隐私政策。

7. Cookie 使用

本服务使用 Cookie 来：

  • 维持您的登录状态
  • 记住您的偏好设置（如主题选择）
  • 提供安全保护（如 CSRF 防护）

您可以通过浏览器设置禁用 Cookie，但这可能影响本服务的正常使用。

8. 儿童隐私

本服务不面向 13 岁以下的儿童。我们不会故意收集 13 岁以下儿童的个人信息。如果您发现我们无意中收集了儿童的个人信息，请联系我们，我们将尽快删除相关信息。

9. 您的权利

根据适用的法律法规，您可能享有以下权利：

  • 访问权：查看我们持有的您的个人信息
  • 更正权：更正不准确的个人信息
  • 删除权：删除您的个人信息
  • 撤回同意：撤回您之前给予的同意

如需行使上述权利，请通过本政策末尾的联系方式与我们联系。

10. 数据保留

我们将在实现本政策所述目的所需的期限内保留您的个人信息，除非法律要求或允许更长的保留期限。当您删除账户时，我们将删除或匿名化您的个人信息。

11. 国际数据传输

如果您使用的 API 提供商位于其他国家/地区，您的对话内容可能会被传输到该国家/地区进行处理。我们会确保此类传输符合适用的数据保护法律。

12. 隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并注明最新更新日期。我们建议您定期查看本政策以了解最新信息。

13. 联系我们

如果您对本隐私政策有任何疑问、意见或建议，请通过以下方式与我们联系：

  • GitHub Issues：https://github.com/WONGIII/LiteWebChatUI/issues

14. 适用法律

本隐私政策受中华人民共和国法律管辖并按其解释。
